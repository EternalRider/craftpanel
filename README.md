## 使用方式（中文）
首先需要新建元素配置面板，拖动物品至左侧配置为元素，拖动物品和元素至中间的上下框以配置该物品的元素成分。

然后新建调和类型面板，在其中添加并配置配方和插槽。

选中角色，点击制作，即可开始使用材料进行合成。

支持许多详细的配置项。

详细配置方式可以查看wiki。

# craftpanel (English)
Craft items using elemental components instead of concrete materials.

### Quick demo
https://github.com/user-attachments/assets/33cc8485-8c8b-4198-872e-53199f3e5c7f

## How to use (English)
1. Create an Element Configuration panel. Drag items into the left column to register them as elements. Then drag items and elements into the center boxes to configure the elemental composition of each item.
2. Create a Craft/Blend/Cook/Forge/Enchant panel (a crafting type). Add and configure recipes and slots inside that panel.
3. Select an actor, open the crafting panel, place materials into slots, and click "Craft" to perform crafting using the configured elemental components.

Features:
- Supports multiple crafting types (blend, cook, forge, enchant) with flexible slot and recipe configurations.
- Scripts and custom requirements for unlocking, crafting, refreshing, and more.
- Per-player recipe unlocking and permission control.

For detailed configuration and examples, please consult the module wiki.

If you need an English translation of the UI strings, see `lang/en.json`.
